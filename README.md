## Original Sources
* **Sources**:
  * **International standards**:
    * International Standard Statistical Classification of Fishery Vessels by Vessel Types (ISSCFV).
    * International Standard Statistical Classification of Fishing Gear (ISSCFG).
    * FAO Major Fishing Areas.
  * **Supranational data sources**:
    * Official Journal of the European Union.
    * European Fleet Register.
  * **National data sources**:
    * Registro General de la Flota Pesquera (Spain).
    * Boletín Oficial del Estado (Spain).
    * Instituto Geográfico Nacional (IGN) - SIGNA tool (Spain).
    * Ministerio de Agricultura, Pesca y Alimentación (Spain). 
* **Document type**: .csv and .xlsx files (e.g. vessels, ports, fishing areas...)
* **Geographical scope**: Europe
* **Temporal coverage**: 2005-2025
* **Language**: English, Spanish

The sources contains documentary records describing the eurpean fishing fleet, infrastructures, administrative divisions and regulatory/international standards.
